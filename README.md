# Octavio AI WhatsApp

🤖 **Octavio AI WhatsApp** est un bot intelligent qui fonctionne uniquement en **message privé** sur WhatsApp.  
Il utilise une intelligence artificielle pour répondre automatiquement aux messages des utilisateurs qui ont activé leur bot via notre site.  

Ce projet est conçu pour :  
- Fournir une **expérience interactive privée** sur WhatsApp  
- Permettre aux utilisateurs de **générer leur code alphanumérique unique** pour activer le bot  
- Permettre à l’administrateur de suivre les utilisateurs et leurs interactions via un **dashboard simple et futuriste**  

Le bot est **facile à installer et à déployer**, et tout est pensé pour fonctionner **en local ou sur un serveur distant** avec Node.js.  

---

## Fonctionnalités principales

- Activation du bot via un **code unique alphanumérique** de 8 caractères  
- Réponses IA automatiques **en privé**, jamais dans les groupes  
- Effet **machine à écrire** pour afficher le code sur le frontend  
- Dashboard admin pour suivre **utilisateurs et messages**  
- Style futuriste noir + vert pour une expérience immersive
